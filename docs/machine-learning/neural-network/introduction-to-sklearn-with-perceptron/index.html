Scikit-learn Intro: Using Perceptron

The name Scikit-learn comes from SciKit (SciPy Toolkit), a third-party extension toolkit for SciPy. After years of development, Scikit-learn has become one of the most popular machine learning libraries. This article takes the classification of the Iris data set as an example to introduce it.

As introduced on the official website, the following are the features of scikit-learn:

  • Simple and efficient data mining and data analysis tools
  • Can be used by anyone and reused in any environment
  • Based on NumPy, SciPy, and matplotlib
  • Open source, available for commercial use (BSD license required)

For the installation of scikit-learn, please refer to this document.

Load Data​

Load the Iris dataset from scikit-learn, where the third column represents petal length* and the fourth column represents petal width*. The classes have been converted to integer labels, where 0 = Iris-Setosa*, 1 = Iris-Versicolor*, 2 = Iris-Verginica_.

from sklearn import datasets
import numpy as np

iris = datasets.load_iris()
X = iris.data[:, [2, 3]]
y = iris.target

print("Class labels:", np.unique(y))  # output: Class labels: [0 1 2]

Preprocessing​

The 150 data sets are divided into 70% training set and 30% through the function sklearn.model_selection.train_test_split Test set:

from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=1, stratify=y
)

Here stratify=y means to divide the data set according to the label value. After the division, the labels of the training set and the test set have the same proportion. You can use np.bincount to print the number of each type of labels:

print("Labels counts in y:", np.bincount(y))  # output: Labels counts in y: [50 50 50]
print(
    "Labels counts in y_train:", np.bincount(y_train)
)  # output: Labels counts in y_train: [35 35 35]
print(
    "Labels counts in y_test:", np.bincount(y_test)
)  # output: Labels counts in y_test: [15 15 15]

Standardize features through the function sklearn.preprocessing.StandardScaler:

from sklearn.preprocessing import StandardScaler

sc = StandardScaler()
sc.fit(X_train)
X_train_std = sc.transform(X_train)
X_test_std = sc.transform(X_test)

Model Training​

Use the neuron model sklearn.linear_model.Perceptron to train fit on the training set:

from sklearn.linear_model import Perceptron

ppn = Perceptron(max_iter=40, eta0=0.1, random_state=0)
ppn.fit(X_train_std, y_train)

Model Evaluation​

After training is completed, predict predict on the test set and print out the number of incorrectly predicted samples:

y_pred = ppn.predict(X_test_std)
print(
    "Misclassified samples: %d" % (y_test != y_pred).sum()
)  # output: Misclassified samples: 3

The training accuracy can be obtained through sklearn.metrics.accuracy_score:

from sklearn.metrics import accuracy_score

print("Accuracy: %.2f" % accuracy_score(y_test, y_pred))  # output: Accuracy: 0.93

Each classification function in scikit-learn has a method score for calculating accuracy. When executed, accuracy_score will be calculated and returned. The output results are the same:

print("Accuracy: %.2f" % ppn.score(X_test_std, y_test))  # output: Accuracy: 0.93

Plot classification boundary​

Plot the algorithm training process:

from matplotlib.colors import ListedColormap
import matplotlib.pyplot as plt


def plot_decision_regions(X, y, classifier, test_idx=None, resolution=0.02):
    markers = ("s", "x", "o", "^", "v")
    colors = ("red", "blue", "lightgreen", "gray", "cyan")
    cmap = ListedColormap(colors[: len(np.unique(y))])

    x1_min, x1_max = X[:, 0].min() - 1, X[:, 0].max() + 1
    x2_min, x2_max = X[:, 1].min() - 1, X[:, 1].max() + 1
    xx1, xx2 = np.meshgrid(
        np.arange(x1_min, x1_max, resolution), np.arange(x2_min, x2_max, resolution)
    )
    Z = classifier.predict(np.array([xx1.ravel(), xx2.ravel()]).T)
    Z = Z.reshape(xx1.shape)
    plt.contourf(xx1, xx2, Z, alpha=0.3, cmap=cmap)
    plt.xlim(xx1.min(), xx1.max())
    plt.ylim(xx2.min(), xx2.max())
    for idx, cl in enumerate(np.unique(y)):
        plt.scatter(
            x=X[y == cl, 0],
            y=X[y == cl, 1],
            alpha=0.8,
            c=colors[idx],
            marker=markers[idx],
            label=cl,
            edgecolor="black",
        )

    if test_idx:
        X_test, y_test = X[test_idx, :], y[test_idx]
        plt.scatter(
            X_test[:, 0],
            X_test[:, 1],
            c="",
            edgecolor="black",
            alpha=1.0,
            linewidth=1,
            marker="o",
            s=100,
            label="test set",
        )


X_combined_std = np.vstack((X_train_std, X_test_std))
y_combined = np.hstack((y_train, y_test))
plot_decision_regions(
    X=X_combined_std, y=y_combined, classifier=ppn, test_idx=range(105, 150)
)
plt.xlabel("petal length [standardized]")
plt.ylabel("petal width [standardized]")
plt.legend(loc="upper left")
plt.show()
