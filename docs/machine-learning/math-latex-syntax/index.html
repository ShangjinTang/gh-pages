info

You might heard or used KaTeX and MathJax, the relationship between them and LaTeX\LaTeXLATE​X is: LaTeX\LaTeXLATE​X is the syntax, KaTeX and MathJax are common LaTeX\LaTeXLATE​X renderers.

This site is rendered using KaTeX, and the syntax may be slightly incompatible with MathJax.

Superscripts and Subscripts​

  1. Use ^ and _ for superscript and subscript respectively, for example, $x_i^2$: xi2x_i^2xi2​.
  2. By default, superscript and subscript symbols only work on the next group. A group is a single character or content wrapped with ${..}$. That is, if you use $10^10$, you get 101010^101010, and $10^{10}$ is 101010^{10}1010.
  3. Braces can also eliminate ambiguity. For example, $x^5^6$ will get an error. Braces must be used to define the associativity of ^, such as ${x^5}^6$ : x56{ x ^ 5 }^ 6x56 or $x^{5^6}$ : x56x ^ { 5 ^ 6 }x56.

Fractions and Radicals​

Expression of fraction:

  1. First, use \frac for the following two groups a, b. If your numerator or denominator is not a single character, use {..} to group. $\frac {a} {b}$ ：ab\frac {a} {b}ba​

  2. Second, use \over to separate the front and back parts of a group. ${a+1 \over b+1}$ : a+1b+1{a+1 \over b + 1}b+1a+1​.

  3. Radical expressions are expressed using \sqrt. $\sqrt[4]{\frac x y}$ ：xy4\sqrt[4]{\frac xy} 4yx​​

  4. Do not use \frac in exponents or integrals. Using \frac in an exponential or integral expression makes the expression look unclear, so it is rarely used in professional mathematical typesetting. A horizontal / should be used instead, with the following effect:

note
  • †\dagger† is called dagger, which is one of the footnote symbols.
  • Use an asterisk * for the first footnote
  • The second one uses the dagger †\dagger†
  • The third footnote uses the double dagger ‡\ddagger‡

†\dagger† is placed next to the author's name. The specific meaning depends on the magazine. You can usually find additional text descriptions. Possible markings: author affiliation, corresponding author, equal contribution, death of the author, etc... (As far as I know, marking death is rare.) When writing a paper, you should check the requirements of the journal for the use of author markings, which is Important essay format.

Wikipedia translates it as "sword mark", and when placed next to the author's name, it symbolizes that the author has passed away.
