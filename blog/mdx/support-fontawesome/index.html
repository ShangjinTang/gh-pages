支持 Font Awesome

2023年12月13日
Sol
全栈开发者

现在可以在 mdx 中使用 Font Awesome 图标。

<FontAwesomeIcon icon="fa-brands fa-react" /> 这是 React 图标。

<FontAwesomeIcon icon="fa-brands fa-github" size="xl" /> 这是加大的 GitHub 图标。

这是 React 图标。

这是加大的 GitHub 图标。

默认情况下，图标继承其父容器的字体大小。 现代浏览器使用默认字体大小 16px。

要更改字体大小，请在 FontAwesomeIcon 中添加以下 size 属性：

  • 2xs: 0.625em (10px)
  • xs: 0.75em (12px)
  • sm: 0.875em (14px)
  • lg: 1.25em (20px)
  • xl: 1.5em (24px)
  • 2xl: 2em (32px)
  • 1x ~ 10x: 1em ~ 10em
