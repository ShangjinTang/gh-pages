支持 Mermaid 图表

2023年11月27日
Sol
全栈开发者

Mermaid 是一个绘制图表的工具。

请参阅 Mermaid 教程 并使用 Mermaid 实时编辑 来进行绘图。

序列图​

Example Mermaid sequence diagram
```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```

流程图​

Example Mermaid flowchart
```mermaid
flowchart TD
    A[Christmas] -->|Get money| B(Go shopping)
    B --> C{Let me think}
    C -->|One| D[Laptop]
    C -->|Two| E[iPhone]
    C -->|Three| F[fa:fa-car Car]
```
